# New Zealand Website

A static multi-page website that showcases New Zealand’s culture, destinations, history, events, and coastal attractions.

## Overview

This project is an informational front-end website built with HTML, CSS, and JavaScript. It includes themed pages, image assets, and a contact form script.

## Project Structure

- `index.html` – Home page
- `history.html` – History of New Zealand
- `attractions.html` – Popular attractions
- `events.html` – Events and activities
- `beaches-coast.html` – Beaches and coastal highlights
- `contact.html` – Contact page
- `form.js` – Form-related behavior
- `scripts/main.js` – Main site JavaScript
- `css/` – Stylesheets
- `images/` – Image assets used across pages

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in your web browser.

Because this is a static site, no build step is required.

## Development Notes

- Keep styles organized in the `css/` directory.
- Keep page-specific behavior in dedicated JS files where appropriate.
- Optimize images before adding new assets to `images/`.

## Technologies Used

- HTML5
- CSS3
- JavaScript (vanilla)
